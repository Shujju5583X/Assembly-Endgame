# Assembly Endgame

A fun, interactive word-guessing game built with React!
Your main objective is to guess the hidden word within eight tries. Each wrong guess chips away at your computer's "Assembly" of programming languages, bringing you closer to utter frustration. Save the tech stack or face the dreadful *Endgame*!

## 🚀 Live Demo
**https://assemblyendgame124.netlify.app/**

## ✨ Features
- **React 18**: Built with the latest functional React.
- **Vite Setup**: Extremely fast local start and optimized build steps.
- **Victory Confetti**: Celebratory effects when you win!
- **Dynamic CSS Styling**: Clean and responsive design.

## 🛠️ Installation & Setup

If you'd like to run the game locally, follow these steps:

1. **Clone this repository**
   ```bash
   git clone <your-repository-url>
   ```

2. **Navigate into the directory**
   ```bash
   cd "Assembly Endgame"
   ```

3. **Install the dependencies**
   ```bash
   npm install
   ```

4. **Start the local development server**
   ```bash
   npm run dev
   ```

5. **Open your browser** and visit the local port (usually `http://localhost:5173/`).
